# dotfiles

## Requirements
1. Install Homebrew
```
$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

2. Install git
```
$ brew install git
```

3. Clone this repository and Run `make setup`

4. Set gitconfig
```
user.email=nao.7ken@gmail.com
user.name=knanao
ghq.root=~/go/src/github.com
```
